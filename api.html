REST API

All of ImageScale's management features are available through the REST API.

Authentication

Requests are authenticated with HTTP basic authorization using an API Key. You can manage your API Keys in the account management interface.

Register a User

POST /api/1/users HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json, text/plain
{ email: "user@example.com", password: "password" }
  • WHEN the email is already in use THEN
    HTTP/1.1 409 Conflict
    Content-Type: application/json; charset=utf-8
  • WHEN any other error occurs THEN
    HTTP/1.1 500 Internal Server Error
    Content-Type: application/json; charset=utf-8
  • WHEN the email is not in use THEN return 200 with user record.
    HTTP/1.1 200 OK
    Content-Type: application/json; charset=utf-8
    {"user":{"id":"5181F322d771ba464826702","email":"user@example.com"}}
